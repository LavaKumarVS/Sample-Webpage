Overview
This project is a non-responsive webpage that simulates an online grocery store,
using HTML, CSS, and JavaScript. It incorporates various web development concepts,
including HTML structure, the box model, CSS styling, and basic JavaScript interactions.
The goal is to create a modern-looking webpage that showcases what you've learned in web development.


Features
HTML Structure: The webpage follows a well-structured HTML layout, including headings,
paragraphs, lists, and links for a clear and organized content presentation.

CSS Styling: CSS is used to style the webpage, providing visual appeal with fonts,
colors, margins, padding, and other styling properties.

Visual Content: The webpage includes images, icons, and possibly other visual elements,
to make it visually engaging and appealing to users.

JavaScript Interactions: Basic JavaScript interactions may be added for user interactivity,
such as pop-up modals, form validation, or dynamic content updates.
